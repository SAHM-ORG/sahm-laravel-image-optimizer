# Changelog

All notable changes to `sahm/laravel-image-optimizer` will be documented in this file.

## [Unreleased]

## [1.0.0] - 2026-01-12

### Added
- Initial release
- WebP and AVIF image conversion
- Responsive image variant generation
- Imagick and GD processor support
- File-based storage (no database required)
- Blur placeholder generation
- Lighthouse optimization (LCP, FCP)
- Configurable quality presets
- Artisan commands (optimize, cleanup, info)
- Queue support for heavy images
- Caching layer
- Helper functions
- Optional API routes
- Comprehensive test suite

### Features
- 70-80% file size reduction
- Automatic srcset generation
- Smart quality detection
- Hash-based file distribution
- Metadata JSON storage
- CDN-ready architecture
